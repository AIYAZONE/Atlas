## 1. 核心视觉规范（Design System Tokens）

设计语言：**Pro-Dark Industrial** —— 专为深色场景下的电商数据精准呈现而生。

### 1.1 色彩规范 (Color Palette)
| 类别 | 变量名 (Design/Code) | HEX | 应用场景 |
|---|---|---|---|
| 底色 | `Background-0` / `bg-primary` | `#0D0F12` | 画布底层、代码编辑器背景 |
| 层级 | `Background-1` / `bg-secondary` | `#1A1D21` | 主界面背景、侧边栏、属性面板 |
| 卡片 | `Surface-2` | `#262A2E` | 卡片背景、弹窗背景 |
| 品牌 | `Primary` / `brand-blue` | `#2D60FF` | 核心按钮、选中高亮、进度条 |
| 点缀 | `Accent` / `brand-orange` | `#FF6B00` | Jackery 品牌点缀、重要警告 |
| 成功 | `Success` / `status-success` | `#00E676` | 构建成功、支付完成、Live 状态 |
| 文本 | `Text-Primary` | `#FFFFFF` | 标题、正文 |
| 文本 | `Text-Muted` | `#9BA1A6` | 标签、占位符、非激活态 |
| 边框 | `Border-Thin` | `#2A2E33` | 分割线、表单边框 |

### 1.2 字体规范 (Typography)
| 类别 | 字体栈 | 说明 |
|---|---|---|
| **Sans-Stack** | `Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif` | 全局 UI 文本 |
| **Mono-Stack** | `'JetBrains Mono', 'Fira Code', monospace` | 代码编辑、日志、技术参数 |

| 层级 | 规格 | 用途 |
|---|---|---|
| H1 | 24px Semi-bold | 页面标题 |
| H2 | 18px Semi-bold | 模块/Section 标题 |
| Body | 14px Regular | 正文、属性面板标签 |
| Small | 12px Medium | 小字、状态信息 |

### 1.3 核心组件规范 (Core Components)
#### 按钮 (Buttons)
*   **Primary**: `brand-blue` 背景 + 白字，4px 圆角（保存、发布）
*   **Secondary**: 透明背景 + 边框 + 白字（预览、取消）
*   **Ghost**: 无背景无边框，仅图标或灰字（侧边栏切换、删除）

#### 输入控件 (Input Controls)
*   **Input / Select**: `bg-primary` 背景，1px solid `border-thin`，高 36px
*   **Switch**: 宽 24px，激活时背景 `brand-blue`

#### 编辑器画布 (Editor Canvas)
*   **选中框**: 2px solid `brand-blue`，右上角浮动操作标签
*   **占位线**: 2px dashed `brand-blue`

### 1.4 间距与栅格 (Spacing)
*   **基础间距**: `4px` (Scale: 4/8/12/16/24/32)
*   **圆角**: `4px` (Soft-Industrial)
*   **面板宽度**: 左侧导航 `64px`，组件面板 `240px`，属性面板 `300px`

## 2. 交互流程资产（Core Interaction Assets）

MVP 链路拆分为 4 大高保真原型模块：

| 模块 | 关键资产内容 |
|---|---|
| **建站引导流**<br>*Onboarding Flow* | • Shopify API 连接验证<br>• 站点身份初始化<br>• 全局组件（Header/Footer）预览确认 |
| **内容生产引擎**<br>*Visual Editor* | • 单层级图层管理（Layers）<br>• 组件库预览选择器（Add Section）<br>• 动态数据选择器（Shopify Data Picker）<br>• 响应式切换预览 |
| **高级配置工具**<br>*Advanced Panel* | • 集成式代码编辑器（语法高亮）<br>• CSS / JSON 切换标签<br>• 高保真色彩选择器面板 |
| **站点管理中心**<br>*Site Admin* | • 站点概览 Dashboard（含构建任务监控）<br>• 页面 / 资产管理（支持批量操作条）<br>• 极简订单同步列表 |

---

## 3. 关键交互模式规范（Interaction Patterns）

| 模式 | 规范说明 |
|---|---|
| **全屏切换**<br>*Mode Switching* | 编辑 ↔ 预览采用 `cubic-bezier(0.4,0,0.2,1)` 过渡，时长 `300ms` |
| **批量操作**<br>*Bulk Actions* | 当选中项 > 0，底部悬浮操作条（Floating Action Bar）向上平滑滑入 |
| **数据绑定反馈**<br>*Dynamic Sync* | 已绑定 Shopify 数据字段带 **Sync** 徽标，悬停显示来源商品名称 |
| **发布状态监控** | • 单页发布：页面顶部进度环<br>• 全局发布：日志控制台实时滚动 Monospace 日志流 |
