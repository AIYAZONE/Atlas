# 管理层决策摘要（Jackery Atlas）

## 决策结论

* 自研 Headless + 静态站（Vue 3 + Nuxt 3）具备明确降本与资产沉淀价值

* 风险可控且有退出机制：与 Shopify 后端保持兼容，随时可回切原生主题

* 建议按 8 周稳健计划交付单站点 MVP，形成可复用资产与组织能力

## 关键价值与 ROI 要点

* 降本：替代高额 CMS 授权与 Page Builder 插件；SSG+CDN 降低托管成本

* 增效：可视化编辑让运营自助发布；性能/SEO 提升带来转化率红利

* 资产：Schema/组件/渲染引擎与适配层沉淀为公司核心数字资产

* 风险项透明：重造轮子、运维与发布延迟已纳入预算与机制（ISR）

## MVP 范围与里程碑（8 周稳健）

* 周 1–2：需求与架构基线（IA、Schema 原则、Monorepo、CI/CD）

* 周 3–4：Builder 与渲染引擎 MVP（编辑/撤销/草稿/发布，JSON→Vue）

* 周 5–6：Site Runtime 与 SEO/性能（Nuxt SSG、Lighthouse≥95、结构化数据）

* 周 7–8：购物车闭环与发布监控（Checkout 跳转、回滚、告警、ISR）

## 风险与退出机制

* 风险：功能迁移重写、运维复杂度、发布延迟

* 应对：预留 30% 重写工时、采用成熟 Serverless 组合、增量构建与窗口管理

* 退出：保持与 Shopify 后端兼容，必要时域名回切原生主题，业务零中断

* 填充 ROI 测算表三情景（保守/基线/激进）并评审

* 按里程碑建立周会推进与验收机制（Checklist）

* 完成 CI/CD、CDN 与备份监控的基础设施落地
