# 文档维护规则（.trae/documents）

## 命名与排序
- 文件名前缀使用两位数字 + 下划线（NN_）表示阅读顺序与分区主题
- 主体使用中文语义名，词间以下划线分隔；避免空格与全角括号
- 新文档示例：07_ContentTypeBuilder_数据流说明.md

## 索引与导航
- 根目录提供 INDEX.md，分组列出所有文档与快速入口
- 文档之间的交叉引用统一使用相对路径与文件名链接

## 新增文档流程
- 复制 TEMPLATE.md，填写标题、分组、版本、维护人、最后更新日期、状态与关联链接
- 按命名规则保存文件，并在 INDEX.md 添加分组与链接

## 变更与评审
- 对文档进行重命名或新增后，需同时更新 INDEX.md
- 合并前 reviewer 检查命名、分组与索引是否一致
